# 01_Global_Index: Foundational Statistical Dataset

**Dataset Source:** Noor Center for Quranic Studies (مركز نون للدراسات القرآنية)  
**Primary Researcher:** Abdul Razzaq Abawi (عبد الرزاق عباوي)  
**Analytical Framework:** Code-1 (Macky)

## Purpose
This document serves as the **Global Index** and the primary empirical baseline for the mathematical research in this repository. All **Code-1** and **Quantum Manifold** calculations are derived from this independent, scholar-verified record.

## Dataset Structure
The index provides a verse-by-verse breakdown:
* **Vers:** Cumulative verse count.
* **SURA (سورة):** Chapter designation.
* **VERSE (آية):** Verse number.
* **WORDS (كلمات):** Word count per verse.
* **LETTERS (حروف):** Letter count per verse.
